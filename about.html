CA Maker allows you to build and simulate your own elementary cellular automaton. A cellular automaton (CA) is a group of cells in a particular state (in this case, black or white). Over time, the state of a cell may change, depending on the state of its neighbors, according to a set of rules. For example, "A white cell that has a black cell on each side should change to black." A complete set of rules will account for what happens to a cell, whatever state its neighbors are in.

CA Maker uses the p5.js JavaScript library for drawing the animations and illustrations.

To learn more about implementing cellular automata (or other natural phenomena) in code, check out The Nature of Code, by Daniel Shiffman.
