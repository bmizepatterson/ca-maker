CA Maker allows you to build and simulate your own elementary cellular automaton. A cellular automaton (CA) is a row of cells in a particular state (in this case, black or white). Over time, the state of a cell may change, depending on the state of its neighbors, according to a set of rules. In the animation below, each row represents one "generation" of the CA.

There are 256 ways for a CA such as this one to behave, and they're numbered 0-255. (Learn more about cellular automata rules.) Change the rule by making a selection below. You can even pause the animation and create each successive generation one click at a time.

Some rules produce more interesting patterns than others. Be sure to try rules 62, 102, 149, and 225. Can you find any other interesting ones?
